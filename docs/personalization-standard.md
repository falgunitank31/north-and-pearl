# Personalization Standard

Personalized product workflows must be product-specific and preserve customer input.

## Requirements

- Show only relevant fields for the product.
- Clearly mark required fields.
- Enforce character limits where known.
- Prevent unsupported symbols when applicable.
- Preserve personalization as Shopify cart line-item properties.
- Display personalization in cart and order details.
- Prevent add to cart if required details are missing.
- Keep fields usable on mobile.

Do not use vague wording like "personalization may be available" on products that need a concrete personalization workflow.
