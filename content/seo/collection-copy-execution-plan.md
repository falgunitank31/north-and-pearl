# Collection Copy Execution Plan

Date: 2026-07-20

Scope: claim-safe organic collection copy for the clean active catalog. This plan is implementation-ready for Shopify collection descriptions, lower SEO sections, and FAQ blocks, but should not be published without orchestration.

## Claim Safety Rules

- Do not state exact materials, plating, stone composition, hypoallergenic status, waterproof status, tarnish resistance, nickel-free status, lead-free status, cadmium-free status, origin, handmade status, sustainability, or delivery timelines unless verified and approved.
- Use `birthstone-inspired`, `crystal-style`, `opal-style`, `pearl-style`, `gold-tone`, or `warm-toned` when describing visual style without confirmed material facts.
- Use `curated edit` for Best Sellers until actual sales or conversion data confirms true best-seller status.
- Use `gift-ready` only as presentation/merchandising language, not as a packaging guarantee unless packaging is confirmed.
- For personalization collections, use `when available`, `where supported`, and `review product details before checkout`.
