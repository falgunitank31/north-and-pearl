# Homepage Foundation

## Branch

`agent/website-architect-homepage-foundation`

## Purpose

Establish the first North & Pearl homepage structure using native Dawn Online Store 2.0 sections.

## Scope

This foundation uses existing Dawn sections only:

- Announcement bar
- Image banner hero
- Multicolumn trust bar
- Featured collection
- Collection list category tiles
- How personalization works
- Gift by occasion
- Image with text brand story
- Placeholder review section
- Email signup banner

## Safety Notes

- No Shopify store connection was made.
- No live theme was modified.
- No theme was published.
- No custom material claims were added.
- Review copy is clearly marked as placeholder content.
- Collection handles are planned handles and must be created in Shopify before launch.
