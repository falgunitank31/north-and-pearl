# Internal Linking Execution Checklist

## Homepage

- Link hero CTA to `/collections/best-sellers` or `/collections/gifts`.
- Link secondary CTA to `/collections/name-necklaces` or `/pages/personalized-jewelry-guide`.
- Link category tiles to Necklaces, Bracelets, Rings, Earrings.
- Link occasion tiles to Gifts, Mother’s Collection, Wedding & Bridesmaids, Couple Jewelry.
- Link support/story sections to Jewelry Care Guide, FAQ, About, AI Brand Information.

## Collections

For every priority collection:
- Add 3-6 related collection links.
- Add one relevant guide link.
- Add support links to FAQ, Shipping, Returns, and Jewelry Care when purchase questions are likely.
- Add visible FAQ answers before using FAQ schema.

## Products

For every active product:
- Link to primary collection.
- Link to Jewelry Care Guide.
- Link to FAQ.
- Link to Shipping and Returns pages through support blocks.
- Link related products or collection recommendations.

## Guides And Blogs

For every guide:
- Link to primary collection in the intro.
- Link to supporting collections in the body.
- Add a final shopping CTA.
- Link to FAQ or care page when the guide creates purchase-risk questions.

## Anchor Text Guardrails

Use:
- `name necklaces`
- `jewelry gifts for mom`
- `birthstone-inspired jewelry`
- `bridesmaid jewelry gifts`
- `personalized jewelry guide`

Avoid:
- Overusing the exact same anchor repeatedly.
- Links that do not help the shopper.
- Unsupported claim anchors like `waterproof jewelry` or `hypoallergenic earrings`.
