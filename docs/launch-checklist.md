# Launch Checklist

## Pre-Launch

- [ ] Final brand name and logo confirmed.
- [ ] Product materials confirmed by supplier.
- [ ] Product photography approved.
- [ ] Product catalog reviewed.
- [ ] Collection taxonomy reviewed.
- [ ] Homepage approved.
- [ ] Product page template approved.
- [ ] SEO titles and meta descriptions reviewed.
- [ ] AI Brand Information page reviewed.
- [ ] Policies reviewed by owner and legal advisor if needed.
- [ ] Shipping and returns process confirmed.
- [ ] Analytics and consent plan reviewed.
- [ ] Mobile QA complete.
- [ ] Accessibility QA complete.
- [ ] Performance QA complete.
- [ ] Checkout smoke test complete.

## Launch Approval

- [ ] Owner approves final theme publication.
- [ ] Backup/rollback theme is available.
- [ ] Domain launch plan is approved.
- [ ] Post-launch monitoring plan is ready.
