# SEO, AEO, and GEO Standard

Organic work must support search visibility, AI understanding, and conversion.

## Required Checks

- One clear primary intent per page.
- Accurate title, H1, headings, meta description, and internal links.
- Visible FAQ content before FAQ schema.
- Product, Organization, Breadcrumb, Article, and FAQ schema contain only verified facts.
- Collection SEO copy supports shopping and does not push products too far down.
- AI Brand Information, About, policies, footer, and schema use consistent brand facts.
- No keyword stuffing, duplicate weak copy, or unverified claims.
