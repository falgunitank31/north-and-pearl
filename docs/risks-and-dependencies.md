# Risks And Dependencies

## Current Risks

- Shopify CLI is not installed locally.
- Shopify GitHub integration status is unknown.
- Product materials and supplier claims are unconfirmed.
- Fulfillment timelines are unconfirmed.
- Live theme publication must be carefully gated.

## Dependencies

- Owner must confirm Shopify development theme strategy.
- Owner must confirm Dawn as the baseline theme.
- Owner must confirm product/material facts before they are used as claims.
- Owner must review policy drafts before publication.
- Owner must approve any app installs, domain changes, tracking scripts, or publishing.

## Mitigations

- Use Git branches and worktrees.
- Use unpublished Shopify development theme.
- Keep all claims configurable and documented.
- Use Git tags for stable milestones.
- Never delete the existing Shopify theme.
